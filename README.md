## Topics 

-Linux Commands


### $ ls    --> list files and directorys in the current directory.   
### $ pwd   --> Prints current directory path.
### $ cd    --> Changes the current working directory.
### $ mkdir  --> Creates a new directory.
### $ rmdir  --> Deletes empty folders from the system hierarchy.
### $ touch -->  Generates a blank file or updates existing file timestamps.
